Annotation
==========

.. default-role:: code

Once you have identified which samples need labels — or which existing labels
need correction — it is time to annotate. FiftyOne's annotation integration
lets you send samples to annotation and bring labels back into your dataset
without leaving your workflow.

Annotation fits naturally into the data-centric loop: curate the samples that
most need labeling, annotate them, then return to :ref:`Explore & Curate
<user-guide-explore-curate>` to assess label quality before training.

**In this section:**

- :doc:`Annotating datasets <annotation>` — Use FiftyOne's built-in in-App
  annotation tools or integrate with external annotation platforms to add and
  edit labels on your datasets.
