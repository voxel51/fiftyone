FiftyOne Fundamentals
=====================

.. default-role:: code

Before diving into the ML lifecycle, these pages cover the core building blocks
of FiftyOne that everything else depends on.

A |Dataset| is the central data structure — it holds your samples, labels,
predictions, and metadata in a unified interface that you can query,
manipulate, and visualize. Understanding how datasets and samples work will
make every other part of FiftyOne feel intuitive.

**In this section:**

- :doc:`FiftyOne basics <basics>` — An overview of datasets, samples, fields,
  labels, and tags. Start here if you are new to FiftyOne.
- :doc:`Using datasets <using_datasets>` — A deep dive into creating, loading,
  modifying, and querying datasets programmatically.
